# Software_Engineering 期中報告 鄭宇彤 黃靖雅
## 主題：Python+ChatGPT制作一个AI实用百宝箱
利用 Python 與 Streamlit 建立一個簡易的「AI 工具箱」網站，實際體驗 AI 聊天與 AI 繪圖的運作方式。

## 一、測驗環境與技術說明

以Python 作為主要語言，利用 Streamlit 建立互動網頁介面，並整合兩項 AI 功能：

- AI 聊天（Ollama 模型）

- AI 繪圖（Pollinations API）

### 使用模組與功能：
模組名稱      	        功能說明               	備註

streamlit           	建立互動式網頁	      簡單、反應即時

streamlit_option_menu	製作側邊導覽選單	    改善操作體驗

subprocess	          呼叫外部 AI 模型	    執行 Ollama

requests	            發送網路請求	        連線 Pollinations API

PIL	                  處理與顯示圖片	      顯示 AI 生成結果

## 二、測驗內容與操作結果
### (一)簡介頁面

這部分讓使用者了解測驗的整體架構與執行方式。

### (二)AI 聊天測試（Ollama 模型）

在聊天功能中，輸入問題文字後，程式會使用 subprocess.run() 呼叫 Ollama 執行 llama3 模型，並回傳文字結果。

### (三)AI 繪圖測試（Pollinations API）

在繪圖功能中，輸入描述文字（如「一隻貓在月球上」），系統會向 Pollinations API 發送請求並取得圖片。

### (四)綜合上述測試觀察重點

測試觀察：

1.支援中文輸入與基本對話。
2.圖片能成功生成且載入速度快。
3.若輸入過長或特殊字元，API 可能會回傳錯誤圖片或失敗訊息。
4.不需要申請 API Key 即可執行。

執行重點：

1.由於 Ollama 屬於本地 AI 模型，它的運算完全在電腦端完成，不會外傳資料，能確保隱私與安全。
不過它需要安裝在正確的路徑下，否則系統會出現「找不到執行檔」的錯誤。

測試觀察：

2.Pollinations 是開放的免費 API，輸入的 prompt 會直接轉換成圖像生成請求。
圖片透過 BytesIO 物件讀入後，以 PIL 顯示在 Streamlit 頁面上。

## 三、測驗介面與實際呈現

測驗的介面設計簡單清楚，左側為選單，右側為功能頁面。
整體配色以白色背景搭配綠色按鈕與圖，讓操作更直覺。

在實際操作中：

1.點選「AI 聊天」可即時與模型互動。

2.點選「AI 繪畫」可根據輸入內容快速生成圖片。

3.整體操作流程順暢，功能之間切換反應良好。

## 四、困難與改善說明
### 1. Python 環境設定問題

在進行測驗前，電腦並未內建 Python，因此需要另外安裝。

實際過程：

從官方網站 https://www.python.org 下載安裝。
安裝完成後，再以 pip install 安裝所需套件（如 streamlit、requests、Pillow 等）。

測試結論：

正確安裝後，整體執行穩定，streamlit run test.py 可順利開啟網頁測試畫面。

### 2. API Key 的問題

多數 AI 服務（例如 OpenAI 或 Gemini）需要 API Key 才能使用。
在測驗初期，這造成一些限制，因為：

1.註冊需信用卡或手機驗證。

2.金鑰可能失效或受限。

3.容易輸入錯誤格式。

實際解法：

選擇使用 Pollinations API 作為替代方案，因為它不需金鑰即可測試，操作上更簡單。
雖然功能比不上付費平台，但足以完成 AI 繪圖測試。

### 3. 聊天程式執行困難
在完成繪圖程式設定後，發現 AI 聊天功能 原本需要使用 API Key 才能連線運作，導致無法順利執行。

問題說明：

1.聊天模組原始版本需透過雲端 API 驅動。

2.因未申請 API Key，執行時出現授權錯誤訊息。

實際解法：

改用 Ollama 的本地程式語法 取代雲端聊天 API，直接透過系統命令執行 Llama3 模型生成回答，成功解決聊天功能無法啟動的問題。
最終測試結果正常，可離線對話。

## 五、測驗結論

這次測驗成功驗證了 Python 與 Streamlit 能夠整合 AI 聊天與 AI 繪圖功能。
透過簡單的網頁介面即可體驗兩種不同類型的 AI 互動：

- Ollama：負責文字回覆。

- Pollinations：負責圖片生成。

整體執行結果良好，雖然過程中需解決環境安裝與執行檔設定問題，但最終能完整體驗 AI 的運作原理。
